# AStarPathfinding 模块 — 编码约定

> 写 `Assets/Scripts/AStarPathfinding/` 下任何代码时自动加载。

---

## 核心原则

1. **纯 C#，零 Unity 依赖** — 不引用 `UnityEngine`，可在控制台测试
2. **Burst 兼容** — 数据结构用 `NativeArray`，不分配托管对象
3. **池化复用** — Grid/Cell 用 Epoch 方案 O(1) 重置，不反复 new

---

## 关键文件

| 文件 | 作用 |
|------|------|
| `AStarSearch.cs` | A* 搜索实现（8 方向，Octile 启发式） |
| `Grid.cs` / `IGrid.cs` | 格子数据结构和状态 |
| `Cell.cs` | 节点（g/h/f + Parent） |
| `FastPriorityQueue.cs` | 优先队列 O(log n) |
| `Vector2Int.cs` | 自定义二维坐标（避免 Unity 依赖） |

---

## 性能要点

- **Grid Epoch O(1) 重置** — 不遍历全部 Cell，用 epoch 标记判断是否属于当前搜索
- **防对角穿墙** — 8 方向移动时检查相邻两格
- **Burst IJob** — A* 核心通过 `BurstAStarJob` 在 Burst 编译下执行

---

## 禁止事项

- ❌ 引用 `UnityEngine` 命名空间
- ❌ `new Cell[gridSize]` 每次搜索 — 用池化
- ❌ 修改 `FastPriorityQueue` 的堆实现 — 已验证性能
